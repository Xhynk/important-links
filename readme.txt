Easily keep track of important URLs inside your WordPress Dashboard

== Description ==

To make management of multiple marketing accounts, tools, and dashboards a bit easier - just add the links into your Important Links page. Any page that allows framing will be iframed in, all pages will have links to the site as well as the login/dashboard page if available.

== Installation ==

1. Upload the `important-links` folder to your `/wp-content/plugins/` directory.

2. Activate the Important Links" plugin.
